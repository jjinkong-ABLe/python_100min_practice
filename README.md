# Python 100-Minute Practice

Beginner-friendly Python practice code written to follow the main topics from a 100-minute Python basics lecture.

The files are not a verbatim copy of the video. They are original practice examples covering the same beginner Python ideas: values, variables, operators, strings, collections, conditionals, loops, functions, modules, exceptions, files, and small projects.

## Practice Files

- `01_variables_types.py`: values, variables, type conversion, and formatted output
- `02_operators.py`: arithmetic, comparison, logical, and membership operators
- `03_strings.py`: indexing, slicing, string methods, and formatting
- `04_collections.py`: lists, tuples, sets, and dictionaries
- `05_conditionals.py`: if, elif, else, and nested conditions
- `06_loops.py`: for loops, while loops, break, continue, and enumerate
- `07_functions.py`: parameters, return values, default values, and keyword arguments
- `08_modules.py`: importing built-in modules and writing reusable helper functions
- `09_exceptions.py`: try, except, else, finally, and safe conversion
- `10_file_io.py`: writing and reading a small text file
- `11_mini_projects.py`: two tiny practice programs using the basics together
